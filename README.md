# Steganography

## Overview
This project implements image steganography using C programming. It hides secret text inside a BMP image and retrieves the hidden message without visibly altering the image.

## Features
- Encode secret message into a BMP image
- Decode hidden message from the encoded image
- File handling
- Command-line execution
- Error handling

## Technologies Used
- C Programming
- File Handling
- Bit Manipulation
- Linux
- GCC

## Files
- main.c
- encode.c
- decode.c
- common.c
- *.h header files
